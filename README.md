# Ecommerce-backend

#### This is a mysql database and backend application for E-commerce site.

### Installation
##### npm install

### Tests
    * Connect to mysql
    * Run source database_name
    * then Run npm start

### Usage
##### Using Insomnia, you can get, post, put and delete data in ecommerce database.
